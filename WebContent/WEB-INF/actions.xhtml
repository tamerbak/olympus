Actions Définition des calcules sur la base des scripts ProtoLANG

Paramètre

Dans le contexte d'un processus métier vous pouvez spécifier des paramétres généraux qui seront utilisés pendant le déroulement de ce denier.

Cela signifie que l'application cherchera la valeur de ce champs dans les paramètres existants lors de l'exécution

Si l'exécution ne se fait pas dans le cadre d'un processus l'appication demandera la saisie des paramètres

Si vous ne voulez pas remplire un champs suite à l'exécution de l'action laissez le champs des paramètres vide
Aide
Les actions aparaissent comme des boutons sur les écrans. Generium détectera directement les arguments en question. Dans le cas où il a besoin de paramétres externes Generium affichera une boite de dialogue pour la saisie de ces paramétres
Après la saisie de la formule de calcule, vous devez renseigner l'action de sauvegarde et n'oubliez pas de séléctionner le champs cible du calcul
